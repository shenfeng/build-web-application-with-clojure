# 《Clojure Web 编程》

想法是学习[build-web-application-with-golang](https://github.com/astaxie/build-web-application-with-golang), 写类似的一本，语言为Clojure。让更多的人了解Clojure Web编程，推广Clojure。

需要介绍的：

* Clojure语言入门：简介，开始，IDE配置，lein等等
* HTTP 简介
* Ring & compojure 简介
* 用Ring抽象HTTP
* 用compojure做路由
* 模板，写出一个hello world
* Session管理
* 数据库操作
* 单元测试
* 打包部署

还在构思当中，欢迎提意见。
